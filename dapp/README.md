# Vault.Stake

## Project info

**UI**: Vite + React + Tailwind

## How to run

```sh
npm install
npm run dev
```

## Notes

This UI is a clean staking dashboard for the workshop contracts.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy?

Use any static hosting (Vercel, Netlify, or a VPS). This is a Vite app.
